# Small Omega

Small Omega (ω) notation is used to describe a lower bound on the growth rate of a function. Specifically, it indicates that a function _g(n)_ grows strictly slower than another function _f(n)_ as _n_ approaches infinity. This means that for any constant _c > 0_, there exists a value _n₀_ such that *g(n) < c*f(n)* for all *n > n₀*. In simpler terms, *f(n)* is a strict lower bound for *g(n)\*.
